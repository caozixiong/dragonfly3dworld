Quality Control of an Additive Manufacturing Artifact Using Dragonfly - (English)

This video demonstrates how to perform a wall thickness analysis or thickness mesh analysis in Dragonfly software, showing how to prepare a 3D model of an additive manufactured component and generate a color-coded thickness map that identifies thin and thick regions.

Tutorial Content

Introduction to Wall Thickness Analysis [00:00:02]

This tutorial demonstrates how to perform a wall thickness analysis (also called thickness mesh analysis) on a 3D scan. The example uses a benchmark additive manufactured component made of aluminum that contains intentional defects for demonstration purposes.

Initial Data Inspection [00:00:26]

To properly examine a 3D scan before analysis:

  • Scroll through the part with good contrast to identify internal features like channels or trapped powder
  • Use the Windows leveling tool for contrast adjustment
  • Select the auto contrasting box and drag it to enhance contrast in areas of interest
  • Press the Escape key to exit contrast mode
  • Zoom in to evaluate specific areas

Setting Up Proper Orientation [00:00:56]

For better visualization:

  • Switch to quad view to see orthogonal perspectives
  • If the object was scanned at an angle, use the rotation and translate tools to align it
  • Adjust the object within the coordinate system to get square views
  • This allows for easier inspection of specific features

Rendering Options [00:02:05]

The sample part contains specific features like overhangs, walls, hollow cylinders, and solid rods.

  • The default grayscale render may not show details clearly
  • The aluminum preset provides a nice set of default render parameters
  • Adjust shininess as needed for better visualization

Creating a Thickness Mesh [00:02:33]

To generate a wall thickness analysis:

  1. View the object in one of the 2D views
  2. Create a segmentation using define range upper OTSU to identify the material of interest
  3. Clean up the data by removing disconnected particles:
  4. Go to refine region of interest → process islands
  5. Select keep by largest to retain only the main connected section
  6. Simplify the mesh by filling small pores:
  7. Use the fill inner areas 3D function to reduce mesh complexity
  8. Export a thickness mesh:
  9. Navigate to the export section on the left panel
  10. Select normal resolution without downsampling
  11. Apply smoothing once for a cleaner surface render

How Thickness Calculation Works [00:04:22]

The thickness calculation:

  • At each surface point, the algorithm calculates the diameter of a sphere that touches both that point and any other surface
  • This is equivalent to sphere-based thickness volume mapping but only calculates values at surface locations
  • This approach is much faster than full volume mapping, especially for large datasets

Visualizing the Thickness Results [00:05:07]

To view and customize the thickness visualization:

  1. Click the eye icon to make the thickness mesh visible
  2. Switch to 3D view and turn off the volume data display
  3. Adjust camera and lighting settings as needed
  4. Select a color map (temperature mapping is recommended) for clear visualization
  5. The resulting visualization shows wall thickness at each point:
  6. Blue areas indicate thinner regions (approximately 0.5mm in the example)
  7. Red areas show thicker regions (up to 12mm in the example)

Conclusion

The wall thickness analysis provides a clear visual representation of thickness variations across the entire part. This information is valuable for quality control, design verification, and identifying potential weak points in the structure.
