This video demonstrates correlative imaging capabilities in Dragonfly software, showcasing how to import, align, and visualize multiple image channels of a mite scanned at different resolutions while exploring various visualization tools and measurement features.

Tutorial Content

Correlative Imaging in Dragonfly: Working with Multiple Data Sets [00:00:02]

This tutorial demonstrates correlative imaging capabilities in Dragonfly, showing how to import data from different imaging modalities and instruments into the same workspace, then register and align them.

Loading Multiple Data Sets [00:00:52]

To begin, we'll load a sample containing two image channels of the same mite scanned at different resolutions:

  • Import objects by selecting "multimite.ORS" object
  • This contains two scans: a body scan and a head scan
  • The body scan is 13 × 16 × 8 millimeters
  • The head scan is 4 × 4 × 3 millimeters

Aligning Multiple Data Sets [00:01:41]

When working with multiple scans, they may not be perfectly aligned:

  1. Right-click on the head scan and select "Align with current view"
  2. To fine-tune alignment, use the translate/rotate tool
  3. Click on the move tool and use keyboard arrow buttons for small adjustments
  4. Press Escape to exit the translate/rotate mode once alignment is complete

3D Visualization Techniques [00:03:33]

To view both datasets in 3D:

  1. Go to the layout panel and select the quad view
  2. Double-click to get the full view
  3. Adjust the 3D render settings:
  4. Try the "Bones" render preset for better visibility
  5. Enable diffuse lighting and specular lighting
  6. Adjust tone mapping for better contrast

Adjusting Window Leveling [00:05:25]

For better visualization:
- Use log scale to narrow the range for viewing
- Adjust gamma for better texture visibility
- Move the black point closer to the left for better surface detail

Background and Lighting Options [00:06:19]

Enhance your visualization with these settings:
- Change background color (white to black gradient works well)
- Enable Edge Detect for contours around the object
- Adjust shadow options (deepen shadows to show more texture)
- Modify lighting direction to highlight different features

Creating a Virtual Floor Effect [00:08:48]

Add a reflective surface to create artistic visualizations:

  1. Enable the virtual floor option
  2. Set the floor on the desired axis (z-axis works well as a "glass floor")
  3. Adjust the reflective surface properties:
  4. Change the color of the reflective surface
  5. Adjust how far away the surface is
  6. Fine-tune the plane using the sliders on the left

Exporting High-Quality Images [00:12:23]

To export your visualization:

  1. Remove text annotations through Scene View Properties
  2. Right-click and select "Export screenshot of view"
  3. Adjust width (up to 4000 pixels) with height automatically adjusted
  4. Enable alpha background if you want only the object without background
  5. Save as a TIFF file for high-quality results

Visualizing Multiple Datasets Together [00:14:40]

To show both the body and head scans together:

  1. Make the body transparent by reducing solidity and opacity
  2. Apply different color schemes to distinguish between datasets:
  3. Change control points in the gradient to create custom colors
  4. Make the head scan more orange/red to contrast with the body
  5. Adjust lighting to enhance visibility

Using the Clipping Box [00:20:32]

To see inside the object:

  1. Enable the clipping box from Visual Effects
  2. Drag the box to clip open the object
  3. Choose inside or outside clipping depending on your needs
  4. Apply the same clipping box to both datasets or individually
  5. Adjust opacity settings for the clipped parts

This technique can be used to:
- Create cross-section views
- Make videos showing progressive clipping
- Create smaller datasets from regions of interest

Measurement and Annotation Tools [00:23:20]

Points and Measurements

  1. Use the Annotate panel to place points of interest
  2. Name and save point collections (e.g., "Surface Head Locations")
  3. Export point coordinates to CSV files

Ruler Tool [00:24:50]

The ruler tool allows precise measurements:
- Measure distances between features
- Measurements are sub-voxel (not limited to voxel edges)
- View intensity profiles along measurement lines

Additional Annotation Tools [00:31:18]

  • Angle measurement: Measure angles between features
  • Path tool: Create and measure paths along complex features
  • Arrow tool: Add arrows with customizable heads, colors and thickness
  • Rectangle annotation: Define regions for analysis

Intensity Analysis [00:34:34]

With rectangular selections, you can:
- Generate histograms of grayscale distributions
- Copy data to clipboard or save to CSV
- Define regions for further analysis

Advanced Visualization: Intensity Projections [00:35:38]

For enhanced visualization:

  1. Use maximum intensity projection to view features across multiple slices
  2. Adjust the projection thickness using the dotted lines in other views
  3. This technique highlights features like hairs that span multiple slices
  4. Minimum intensity projection is useful for visualizing porosity

Maximum intensity projection is particularly useful for seeing structures that span multiple slices in a single 2D view, making it easier to identify patterns and connections.

Conclusion

Dragonfly offers powerful tools for correlative imaging, allowing you to work with multiple datasets at different resolutions. The software provides extensive visualization options, measurement tools, and analysis capabilities to extract meaningful information from your data. By combining datasets and using the various rendering and annotation features, you can create comprehensive visualizations that highlight the features most relevant to your research.
