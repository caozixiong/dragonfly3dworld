July 2022 Workshop Video 7: Deep learning de-noising using Noise2Void - (English)

This video demonstrates the Noise to Void deep learning denoising feature in Dragonfly software, showing how to train and apply an AI model to effectively remove speckle noise from metal powder CT scan data for additive manufacturing applications.

Tutorial Content

Deep Learning in Dragonfly Software: Noise to Void

Introduction [00:00:03]

This tutorial demonstrates the use of deep learning in Dragonfly software, focusing on the noise to void feature. Deep learning is the strength of Dragonfly software, and this tutorial will show different ways of using artificial intelligence and deep learning to help both new and expert users segment their data more effectively.

Noise to Void Denoising [00:00:59]

The first demonstration is the noise to void feature, which uses deep learning for denoising images. While Dragonfly includes many traditional image filtering tools in the image filtering toolbox, this tutorial focuses on the deep learning approach.

Setting Up the Data

For this demonstration, we're using a noisy dataset of metal powder scans for additive manufacturing. The data shows:
- Speckle noise
- Ring artifacts
- Metal powders with pores inside

Creating Training Data

To use deep learning tools, we need:

  1. A dataset as input
  2. A frame for visual feedback
  3. A mask for training

To create these:

  1. Select Annotate and create a layer
  2. Use the rectangle tool to create a region for visual feedback
  3. Use the segmentation tool paintbrush to paint a mask on specific slices
  4. For this example, we're using two slices for training

Training the Model

  1. Click on the image channel
  2. Go to Artificial Intelligence
  3. Select Custom deep model architecture > Noise to void
  4. Choose to train a new model
  5. Name the model (e.g., "demo powder model")
  6. Set the input as the metal powders dataset
  7. Set the mask as the previously created mask
  8. Set augmentation level
  9. Higher augmentation takes longer but produces better results
  10. Augmentation creates more training data by flipping, sharing, scaling, and adding noise
  11. Set the visual feedback frame
  12. Click Train

During training, you'll see:
- Progress bars
- Visual feedback on the right
- The model's initial loss and loss value on the left
- These values should decrease as the model improves
- The best result plateaus near zero

Applying the Model [00:07:04]

Once the model is trained, you can apply it to the dataset:

  1. Select the input dataset
  2. Apply the trained model
  3. The model will process all slices in the dataset, creating a new denoised dataset

Comparing Results

To compare the original and denoised data:

  1. Use the ruler tool to create a profile across features of interest
  2. Adjust colors to distinguish between original and denoised data
  3. Observe the smoothing of noise while preserving important features

The denoised result shows significantly reduced noise while maintaining the important structural details of the powder particles.

Reusing the Model

An important advantage of this approach is that you don't need to train the model every time. For subsequent powder scans:

  1. Go to the image channel
  2. Select Artificial Intelligence > Apply
  3. Select your previously trained model
  4. Choose the input dataset
  5. Apply the model

This allows you to quickly denoise similar datasets without repeating the training process.

Conclusion [00:11:11]

The noise to void feature in Dragonfly provides an effective deep learning approach to denoise image data. By training a model on a small portion of your data, you can create a powerful denoising filter that can be applied to the entire dataset and reused for similar datasets in the future. This approach preserves important structural details while significantly reducing noise, making subsequent analysis and segmentation more accurate.
