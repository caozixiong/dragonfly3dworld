Intensity Calibration in Dragonfly

Introduction [00:00:03]

This tutorial demonstrates how to perform intensity calibration in Dragonfly. This technique is particularly useful for bone mineral density calibration of grayscales in CT data, but can be applied to other purposes as well. The demonstration uses CT data of a lizard osteoderm, a lizard bone, and a bone mineral density calibration phantom.

The data comes from a published study in the Journal of Anatomy that investigated osteoderm density to determine if these bony plates in lizards can act as calcium reservoirs.

Understanding the Calibration Need [00:00:57]

In 16-bit CT data, gray values typically range from 0 to 65,535. The goal of intensity calibration is to convert these arbitrary gray values to meaningful density measurements. In this example, we know that the bone mineral density calibration phantom has a density of 1200 milligrams per cubic centimeter, which will serve as our reference point.

Creating a Region of Interest [00:01:43]

To calibrate the intensity scale, first create a segmentation of the calibration phantom:

  1. Use the local Otsu brush with a 3D spherical paintbrush to select the dense pixels within the phantom
  2. Ensure you select enough pixels (approximately 10 million voxels in this example) without including other structures
  3. Check the histogram to determine the average gray value of the phantom
  4. In this example, the values are centered around 45,292 (approximately 45,000)

Calibrating the Intensity Scale [00:02:57]

Once you have identified the gray value of your reference material:

  1. Right-click on the image channel
  2. Select "Calibrate Intensity Scale" at the bottom of the context menu
  3. Choose the appropriate calibration unit (Bone Mineral Density comes pre-installed with Dragonfly)
  4. Set up your reference points:
  5. Double-click on the 1200 value (representing 1200 mg/cm³)
  6. Enter the gray value you measured (45,000 in this example)
  7. Click "Calibrate" and close the dialog

Adjusting the Display [00:04:37]

After calibration, the image may initially appear blank because:

  1. The display is still set to show the original range (up to 65,000)
  2. The calibrated values are now centered around 1,000

To fix this:
1. Go to the window leveling tool
2. Click "Reset"
3. The display will adjust to show values from approximately -400 up to 1,900

Verifying the Calibration [00:04:55]

To verify that the calibration worked correctly:

  1. Draw a ruler annotation across both the osteoderm and the calibration phantom
  2. Check the profile intensity
  3. Confirm that the calibration phantom shows an average value of 1,200 as expected
  4. Observe that the osteoderm has a value above 1,200 that drops across its width

Conclusion [00:05:27]

Intensity calibration in Dragonfly allows you to convert arbitrary gray values to meaningful density measurements, which is particularly useful for bone mineral density studies. This technique enables quantitative analysis of CT data by providing standardized density values rather than relative grayscale intensities.
