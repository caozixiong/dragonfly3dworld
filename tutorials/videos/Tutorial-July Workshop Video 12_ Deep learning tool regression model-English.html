July Workshop Video 12: Deep learning tool regression model - (English)

This video demonstrates how to use deep learning as a regression model to replicate traditional image filtering, specifically training a U-Net model to function as a median filter for noise reduction in image data.

Tutorial Content

Using Deep Learning for Image Filtering in Dragonfly

Introduction [00:00:03]

This tutorial demonstrates how to use the Deep Learning tool as a regression learning model to replicate traditional image processing filters. Specifically, we'll train a U-Net model to replicate the effect of a median filter on a noisy image dataset.

Preparing the Data [00:00:28]

We'll start with the denim dataset, which contains some noise that we want to filter out.

  1. First, let's apply a traditional median filter to create our training target:
  2. Navigate to the Workflows menu and select Image Filtering
  3. In the dropdown box, type "median" to select the median filter
  4. Set the kernel size to 5
  5. Apply the filter to all slices

Now we have two image channels:
- The original noisy data (input)
- The median-filtered data (target output)

Creating a Regression Model [00:02:05]

To create a deep learning model that replicates the median filter:

  1. Select the original data
  2. Go to the Deep Learning tool
  3. Create a new model with these settings:
  4. Model type: U-Net
  5. Task type: Regression (not semantic segmentation)
  6. Input count: 1
  7. Dimension: 2.5D
  8. Name: "median filter"
  9. Click Generate to create the model

Creating a Mask for Training [00:02:58]

The training process requires a mask to define the region for training:

  1. Go to the Segmentation tab
  2. Create a new ROI called "mask"
  3. Select a slice
  4. Use the brush tool (make it larger by scrolling while holding Control)
  5. Paint the entire slice to create a mask

Setting Up Training [00:03:34]

To train the model:

  1. Set up the training inputs:
  2. Input: denim input (original data)
  3. Output: denim median filtered image channel
  4. Mask: the mask we just created
  5. Adjust the augmentation settings as needed
  6. Create a preview frame to monitor training progress:
  7. Go to the main context
  8. Select a rectangle annotation to define a preview region
  9. Set this region as the visual feedback frame

Training the Model [00:04:09]

  1. Click on Train to start the training process
  2. The model will learn to transform the input image to match the median-filtered output
  3. Monitor the training progress in the preview window
  4. The training may reach a steady state quickly and perform early stopping

This demonstrates image-to-image regression. If you have a specific filter and examples of before/after images, you could incorporate that filter into Dragonfly indirectly using a deep learning model.

You can save a 15-second video of the training progression for presentation purposes by using the export option.

Applying the Trained Model [00:05:52]

After training is complete:

  1. Save the model
  2. Return to the original data
  3. Select Filter with AI
  4. Choose the trained "unit median filter" model
  5. You can preview the results on a single slice or apply it to all slices

Comparing Results [00:06:20]

To compare the original, traditionally filtered, and AI-filtered results:

  1. Create a ruler annotation across an area of interest
  2. Use the intensity profile tool to visualize the differences between:
  3. Original data
  4. Median filtered data
  5. U-Net median filtered data

The AI-filtered results should be very similar to the traditional median filter, demonstrating successful training of the regression model.

Conclusion [00:08:22]

This tutorial has demonstrated how to use deep learning for image-to-image regression in Dragonfly. This approach allows you to replicate traditional image filters or even create custom filters based on training examples. The technique can be extended to other image processing tasks where you have paired input and output examples.
