Tutorial Content

Deep Learning Segmentation on a Turbine Blade Using Dragonfly

Introduction [00:00:03]

This tutorial demonstrates how to perform deep learning segmentation on a turbine blade using Dragonfly software. When working with 3D CT scans of turbine blades, manual inspection of slices can help identify defects and geometry deviations. However, for quantitative work, segmentation is necessary to compare geometry or export meshes.

Traditional thresholding methods often fail with turbine blades due to beam hardening and image artifacts - selecting too little material near the top and too much material inside curves. A deep learning approach provides a more effective solution.

Loading and Examining the Data [00:00:14]

The tutorial uses a 3D CT scan of a turbine blade visible in the top left of the interface. Cross-sections can be manually inspected, but proper segmentation is required for quantitative analysis.

Creating Training Data [00:01:41]

The deep learning approach requires ground truth slices that have been carefully segmented. These will be used to train the model.

  1. Navigate to slice 1000
  2. Create a region of interest called "blade" (blue color)
  3. Use the smart grid tool (super pixels) which uses a watershed algorithm in 2D
  4. This fits the curve nicely for this particular data
  5. Manual methods and smoothing can also be used to refine the segmentation

Creating a Mask [00:02:38]

First, create a mask to help the deep learning model identify the training area:

  1. Color in the entire slice as the mask
  2. Navigate to slice 500
  3. Use the super pixels tool to highlight the blade
  4. Create a mask for this slice as well
  5. Navigate to slice 1500 for symmetry
  6. Repeat the process of creating a mask and segmenting the blade

Note: While this demonstration uses only three slices for training, providing more data will improve model performance. For production models, spend more time creating precise edges and training with additional slices.

Training the Deep Learning Model [00:04:52]

  1. Open the artificial intelligence deep learning tool
  2. Select "Create a new model"
  3. Configure the model:
  4. Two classes: material and background
  5. 2.5D with three slices input
  6. Name it "turbine blade" with the date
  7. Set up the training parameters:
  8. Image channel as input
  9. Multi ROI as output
  10. Define the mask
  11. Increase augmentation as needed
  12. Enable visual feedback for monitoring

Once configured, start the training process. The model will improve with each epoch:
- First epoch: initial results (not very good)
- Second epoch: noticeable improvement
- Third epoch: further improvement
- Continue for several epochs until satisfactory results are achieved

Applying the Model [00:07:11]

After training completes:

  1. Take the turbine blade input
  2. Apply the trained model to all slices

This process takes longer but is not user-dependent, so it can run overnight in an automated workflow.

Examining and Refining Results [00:07:52]

The segmentation results appear as a blue-purple color overlay on the 3D render:

  1. Click on the segmentation output to examine the blade or its complement
  2. Extract the class as a region of interest
  3. View the segmentation in 3D by switching off the rendering

The segmentation looks good overall, but there may be some loose bits near the bottom of the blade that need refinement:

  1. Refine the region of interest by selecting "Keep the largest"
  2. This removes all small disconnected bits

Creating a Mesh for Export [00:08:54]

To finalize the segmentation:

  1. Generate a contour mesh
  2. This can be exported as an STL file
  3. The mesh provides better visualization of the segmentation results

The final result is a clean, accurate segmentation of the turbine blade that can be used for further analysis or manufacturing.

Conclusion [00:09:42]

This tutorial demonstrated the complete workflow for deep learning segmentation of a turbine blade in Dragonfly:
1. Creating training data on select slices
2. Training a deep learning model
3. Applying the model to the entire volume
4. Refining results and exporting as a mesh

Even with minimal training data (just three slices), the deep learning approach provides excellent results that overcome the limitations of traditional thresholding methods. For production use, more training data and refinement would yield even better results.
