July 2022 Workshop Video 1: Basics of Dragonfly using a lego figurine - (English)

This video demonstrates the basic functionality of Dragonfly software using a micro CT scan of a Lego figurine, covering essential tools for data visualization, manipulation, and analysis including importing image stacks, adjusting contrast, rotating objects, and creating animations.

Tutorial Content

Getting Started with Dragonfly: Basic Navigation and Visualization

Introduction to Dragonfly [00:00:03]

This tutorial is based on a workshop held in Grenoble, France. We'll be exploring data from a micro CT scan of a Lego figurine, provided by Tanya Shikora from the Exelon Academy. We'll start with the basics of Dragonfly version 22.1 and learn how to navigate the workspace.

Understanding the Dragonfly Workspace [00:00:37]

When you first open Dragonfly, you'll see panels on the left and right sides of the interface. These panels can be:
- Rearranged in any order
- Opened or closed based on your preference
- Positioned at the top, bottom, or sides of the window

Working with Sessions and Files [00:01:19]

Dragonfly works with sessions that can contain:
- Multiple datasets and images
- Measurements and regions of interest
- All your work saved in a single session file

This makes it easy to share with colleagues or reopen your work later. You can also save individual objects as separate files if needed.

Importing Image Files [00:01:12]

To load data:
1. Go to File > Import Image Files
2. This can load data from any imaging modality:
- Image stacks
- Micro CT scans from commercial vendors (Zeiss, Xradian, etc.)
- VGI format and many others

Loading and Configuring Image Data [00:02:44]

For this tutorial, we'll import a 40-micron image stack of the Lego figurine:

  1. Select all images in the stack and click Open
  2. Give your dataset a name (optional) - we'll call it "Lego man"
  3. Set the image spacing (voxel size) - in this case 0.04 mm in all three axes (isotropic)

Optional Import Settings [00:03:27]

When importing large datasets, you can:
- Downsample the data by taking every second pixel in X, Y, and Z to reduce file size
- Crop the dataset during import using the cropping tool
- Drag the sliders to define the crop region in X, Y, and Z
- Use the preview to ensure you're not cropping important features

Basic Visualization and Navigation [00:05:23]

After importing, the dataset appears in the workspace as an image channel. You can:
- Toggle visibility using the eye icon
- View in 3D by enabling the 3D visualization button
- Scroll through slices by dragging the slider or using the mouse wheel
- Adjust contrast by left-clicking and dragging in the 2D view

Adjusting Contrast [00:07:02]

Use the Windows Leveling tab to adjust contrast:
- Drag the histogram left/right to adjust brightness
- Narrow the contrast area for sharper contrast
- Adjust gamma for fine-tuning

Navigation Controls [00:07:56]

Basic navigation shortcuts:
- Left and right mouse buttons + drag to move around the object
- Mouse wheel to zoom in/out
- Double-click on any view to maximize it
- Press Escape to return to default mode from any tool

Working with Multiple Views [00:06:25]

Switch to quad view to see multiple perspectives:
1. Click on the quad view button
2. Enable 3D view
3. Left-click and drag in the 3D view to rotate the object

Note that each window has a colored outline when active. Functions only apply to the active window.

Using Crosshairs [00:10:56]

The crosshairs help you locate features across different views:
1. Drag the crosshair to a feature of interest
2. The feature will be centered in all three cross-sectional views
3. Enable "Show 2D View Planes" to see the slice positions in the 3D view

Manipulating the Object [00:12:03]

Flip and Rotate Tools [00:12:06]

Use the flip and rotate tools to:
- Turn the object horizontally or vertically
- Flip the object to face different directions
- Quickly get a nice viewing angle without changing the data

Translate and Rotate Tools [00:15:49]

To reorient your data along specific axes:
1. Go to the Translate Rotate tools
2. Use the rotation controls to align the object as desired
3. This is useful for aligning the object with X, Y, or Z axes

You can also use these tools to align different objects manually.

Advanced Visualization Options [00:24:31]

3D Rendering Settings [00:24:33]

For better 3D visualization:
1. Use the 3D presets for quick rendering options
2. Adjust quality settings and enable anti-aliasing for better images
3. Change the background color or gradient using Scene View Properties

View Settings [00:25:42]

Additional view options:
- Toggle between perspective and orthographic projection
- Enable scale bars (only available in orthographic mode for 3D)
- Move legends and scale bars by dragging them
- Toggle text annotations on/off

Creating Videos and Screenshots [00:28:15]

To create a slice video:
1. Position the view as desired
2. Go to the Animation panel
3. Set the start and end slices
4. Adjust frames per second
5. Click Record to save the video

For screenshots:
1. Set up the desired view
2. Right-click and select "Export Screenshot of View"
3. Choose the format and resolution

Conclusion

This tutorial covered the basic navigation and visualization tools in Dragonfly. You've learned how to:
- Import and configure image data
- Navigate in 2D and 3D views
- Adjust contrast for better visualization
- Manipulate objects using various tools
- Create videos and screenshots

In the next tutorials, we'll explore more advanced features of Dragonfly for image analysis and processing.
