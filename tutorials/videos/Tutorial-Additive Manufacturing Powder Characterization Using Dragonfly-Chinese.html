Tutorial Content

粉末颗粒分水岭分割教程

介绍 [00:00:02]

本教程演示如何对CT扫描数据集中的粉末颗粒进行分水岭分割。当您需要分离数据中相互接触的物体以便单独计算每个颗粒的属性时，分水岭分割特别有用。

数据集概述 [00:00:21]

本演示使用一个小型的、裁剪过的粉末数据集来简化工作流程。虽然我们将从切片视图开始，但我们也会在3D中检查数据以更好地理解颗粒分布。

对于这种类型的分析，建议使用四视图同时查看切片和3D视角。

基本分割 [00:01:04]

在应用分水岭算法之前，我们需要创建颗粒的基本分割：

  1. 导航至Segmentation选项卡
  2. 选择Define Range
  3. 选择Upper OTSU - 一种自动阈值方法，用于将体素分类为材料或背景
  4. 类似地，使用Lower OTSU并Add to Network完成基本分割

此时，您应该有一个简单的分割，包含颗粒和背景，但接触的颗粒仍然会连在一起。

创建种子点 [00:02:11]

分水岭算法需要种子点。有两种方法可以创建这些种子点：

方法1：腐蚀

  • 对分割后的颗粒进行腐蚀
  • 这在大多数颗粒内部创建种子点

方法2：距离图（推荐）

  1. 选择背景
  2. 右键点击并选择Create Distance Map
  3. 这会创建值，其中最高值是离您选择的区域最远的地方
  4. 对此距离图应用Upper OTSU
  5. 手动将阈值稍微向上调整，以确保颗粒不相互接触，同时保留颗粒内的值
  6. 这将创建一个"距离映射颗粒"分割

准备分水岭 [00:03:56]

为准备分水岭算法的输入：

  1. 将背景添加到距离映射颗粒中
  2. 通过将它们组合成一个新的感兴趣区域来创建联合（将其命名为"seed for watershed"）
  3. 关闭其他区域以简化视图，只保留颗粒和背景

为了使分水岭正常工作，标记需要像多ROI一样分离，每个单独的项目有不同的颜色。

计算分水岭 [00:04:45]

  1. 右键点击图像通道
  2. 选择Compute Watershed
  3. 此过程突出显示对象之间的边缘并计算分离的对象

结果将是一个分水岭多ROI，其中每个颗粒都有单独的值。

分析结果 [00:05:14]

一旦分水岭分割完成：

  1. 重命名结果以提高清晰度
  2. 使用Scalar Generator计算颗粒属性
  3. 出于演示目的，计算体积和球形度
  4. 在3D中查看结果
  5. 根据体积或球形度等测量值对颗粒进行分类
  6. 根据需要调整颜色编码（温度色图效果很好）

结论 [00:06:38]

这个工作流程演示了如何在Dragonfly中使用分水岭分割来分离接触的颗粒。虽然还有其他方法可以使用分水岭分割工具，但这种方法提供了一种直接的方式来分离颗粒，以便进行单独分析和属性计算。
