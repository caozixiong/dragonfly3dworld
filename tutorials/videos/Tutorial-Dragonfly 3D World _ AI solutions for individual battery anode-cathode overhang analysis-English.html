Tutorial Content

Battery Cell Analysis in Dragonfly Software

Introduction [00:00:05]

This tutorial demonstrates how to perform analysis of battery cells using Dragonfly software. You'll learn how to:

  • Perform manual measurements easily
  • Conduct typical inspection of cylindrical battery cells
  • Use automation tools with deep learning
  • Train your own model for battery analysis

Understanding the Data [00:02:45]

The data shows a cylindrical battery cell with anodes and cathodes rolled up adjacent to each other, surrounded by a metal casing. The cross-sectional views provide the best perspective for making measurements.

In the 2D cross-sectional view, we can see:
- Cathode materials
- Anode materials
- Spacer materials between them

The scan comes from a reasonably fast scan on a Comet Exelon FF35 system. This demo case shows how scanning and analysis of cells can be fully automated.

Manual Inspection for Defects [00:04:36]

When inspecting battery cells, you need to look for:
- Inclusions which shouldn't be there
- Areas out of specification
- Broken pieces or defects

To better visualize defects, adjust the contrast:
1. Click on the data view
2. Use the window leveling option to change the contrast of the image

Using Cylindrical Cross-Section View

The cylindrical cross-section view allows you to examine the battery at different angles:

  1. Turn around the object to get cross-sections at different angles
  2. When you find an interesting feature, place the crosshair at that point
  3. Examine the feature in other views to confirm the defect

Adding Annotations [00:06:02]

To mark defects for later reference:

  1. Select the Annotate option
  2. Choose Points
  3. Hold down Control and click in the window to create a point
  4. The annotation will show the X, Y, Z location of the defect

To navigate between annotations:
1. Right-click on an annotation in the list
2. Select Center View to focus on that point in all views

You can export annotations to a CSV file:
1. Select all annotations (click first, then Shift+click last)
2. Right-click and select Export to file
3. Save as a CSV file

Manual Measurements [00:09:52]

To manually measure overhang heights:

  1. Use the Ruler tool from the annotation panel
  2. Click to place the start and end points of your measurement
  3. Adjust measurements by selecting and moving the endpoints
  4. Use the Nearest Interpolation mode to see actual pixels for more precise measurements
  5. Use the Floating Caption option to position measurement labels more clearly

To save your measurements:
1. Right-click on the image and export a screenshot
2. Select all rulers and right-click to export data to a CSV file

Automated Analysis with Deep Learning [00:13:02]

Manual measurements can be time-consuming when analyzing multiple cells. Deep learning models can speed up the process by automatically segmenting the regions of interest.

Using a Pre-trained Model [00:14:40]

  1. In the segmentation panel, select Segment with AI
  2. Choose your trained model
  3. Use the Preview option to segment only the current slice
  4. The result will show different classes (e.g., cathodes, background, anode overhang)
  5. Right-click on the result and export the class of interest as an ROI

To analyze the segmentation:
1. Right-click on the exported ROI and select Compute Measurements
2. Choose appropriate measurements (e.g., 2D major axis length for overhangs)
3. View the results with color mapping to visualize differences in measurements

Training Your Own Deep Learning Model [00:17:10]

The Segmentation Wizard in Dragonfly allows you to train models without being an AI expert.

Setting Up a Training Session [00:19:19]

  1. Click on AI Segmentation Wizard and select New
  2. Choose whether to start with a pre-trained model or from scratch
  3. Add your first frame by clicking the plus button
  4. Define your classes (e.g., background and anodes)

Creating Training Data [00:20:14]

For each class:
1. Select the appropriate brush tool (e.g., 2D brush or local Otsu brush)
2. Paint examples of each class in your frame
3. Add more frames from different regions of your sample
4. Set one frame as a monitoring frame to track progress during training

Training the Model [00:21:45]

  1. Click Start Training
  2. Choose a model type or use a pre-trained model as a starting point
  3. Monitor the training progress
  4. Training can take from 20 minutes to several hours depending on your data and computer

Saving Your Model [00:36:37]

When training is complete:
1. Go to the model tab and click the save button to publish it to your system
2. Or exit and choose to publish the model with a custom name

Conclusion [00:37:09]

Dragonfly software provides powerful tools for battery cell analysis, from manual inspection and measurements to automated deep learning approaches. While manual methods provide precision and control, deep learning models can significantly speed up your workflow once properly trained. This combination of techniques allows for comprehensive analysis of battery cells for research and quality control purposes.
