Dragonfly Daily 01: Working with Images in Dragonfly (2020) - Summary and Tutorial

This video introduces the Dragonfly Daily webinar series, focusing on using Dragonfly for scientific imaging tasks. The first episode covers working with image data, including importing various file formats, setting pixel sizes, visualizing data in 2D and 3D, and saving Dragonfly objects.

Tutorial: Working with Image Data in Dragonfly

1. Introduction [00:00:02]

  • The Dragonfly Daily webinar series introduces tutorials on using Dragonfly for scientific imaging.
  • The first topic is working with image data.

2. Accessing Example Data [00:06:25]

  • Navigate to the Digital Rocks Portal to access publicly available datasets.
  • Datasets include TIF stacks, Aviso mesh data, and raw data.

3. Importing a TIF Stack [00:08:15]

  • Go to File > Import Image Files.
  • Click "Add" and navigate to the folder containing the TIF images.
  • Select all TIF files and click "Open".
  • Input the correct pixel size (e.g., 54.1 microns per pixel) [00:09:46].
  • Click "Finish".

4. Visualizing Image Data [00:10:19]

  • Observe the loaded image data, noting the number of slices.
  • Use the layout panel to switch to a multi-view layout for orthogonal and 3D views [00:11:15].
  • Manipulate crosshairs to view specific areas in different views [00:11:48].
  • Double-click a view to make it fullscreen [00:12:21].
  • Zoom using the scroll wheel (depress and drag) or by adjusting the field of view [00:12:36].
  • Pan by pressing and dragging with both mouse buttons [00:13:24].

5. Understanding Pixels [00:13:45]

  • Zoom in to see individual pixels by disabling interpolation in the 2D settings [00:13:57].
  • Use the probe tool to check pixel intensity and coordinates [00:14:36].

6. Importing Raw Data [00:18:18]

  • Go to File > Import Image Files.
  • Click "Add" and select the raw file.
  • Specify the dimensions (e.g., 1024x1024x800) and data type (e.g., 8-bit) [00:19:05].
  • Input the correct pixel size [00:19:29].
  • Click "Finish".

7. Adjusting Brightness and Contrast [00:20:10]

  • Use the Window Leveling panel to adjust brightness and contrast [00:20:17].
  • Manipulate the histogram to set the window for mapping brightness [00:20:35].
  • Alternatively, drag up/down for brightness and left/right for contrast directly on the image [00:21:51].

8. Saving as ORS Object [00:22:47]

  • Right-click the data and select Export > As ORS Object.
  • Save the data in Dragonfly's native format to preserve pixel size and other metadata.

9. Importing an A.M. File [00:24:21]

  • Go to File > Import Images.
  • Select the A.M. file.
  • Set the pixel size [00:24:53].
  • Click Finish.

10. Q&A and Conclusion [00:26:27]

  • Questions about music, video availability, graphics card requirements, MRC files, window leveling, video downloads, multi-channel images, upside-down images, and Dragonfly's strengths are addressed.
  • Dragonfly's flexibility, extensibility, and powerful image segmentation tools are highlighted [00:33:03].
