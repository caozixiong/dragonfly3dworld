Tutorial Content

Cylinder Coordinate Transforms in Dragonfly

Introduction [00:00:32]

In this tutorial, we'll explore how to transform data from a rectangular coordinate system to a cylinder coordinate system in Dragonfly. This allows you to extract and visualize data in a different way, particularly useful for cylindrical samples.

We'll be using Dragonfly 2020.1 Release Candidate 2 and a dry sand pack dataset for this demonstration.

Understanding Cylinder Coordinate Transforms [00:02:19]

The concept behind cylinder coordinate transforms can be thought of as a "soup can unwrapper." Imagine taking a cylindrical object like a soup can, cutting along the seam of the wrapper, and laying it flat on a table. This gives you a rectangular image representing the pixels around the edge of the sample.

This transformation allows you to:
- Inspect the surface of your cylindrical object
- Examine the object at subsurface levels at various radii
- Transform data from cylinder coordinates to rectangular coordinates

Required Tools [00:03:04]

To perform this transformation, you'll need:
- An image dataset
- A cylinder object
- The "Extract Soup Can Wrapper" extension from the Infinite Toolbox

Note that this functionality is not part of the standard Dragonfly installation but is available as a custom menu item that anyone can download from the Infinite Toolbox.

Basic Extraction Process [00:03:27]

  1. Create a cylinder of any height and diameter, setting parameters interactively
  2. Select the image data
  3. Control-click to also select the cylinder
  4. Right-click and select "Execute Macro Extract Soup Can Wrapper"

The extraction is performed in Python, so it may not be especially fast. A compiled version is planned for future releases.

Viewing the Results [00:04:27]

After running the extraction, you'll see two different images created. These images are positioned at the same origin in the world coordinate system, essentially superimposed on each other.

To analyze data at multiple radii, you can use a macro that:
- Runs the extraction at one radius
- Changes the radius and runs again
- Applies displacements to separate the images visually

Using the Radius Range Macro [00:05:58]

  1. Go to Utilities > Macro Player
  2. Select "Soup Can Wrapper Over Radius Range"
  3. Select both the image and cylinder
  4. Click "Play"
  5. Enter the starting radius (e.g., 1 millimeter)
  6. Enter the ending radius (e.g., 1.5 millimeters)
  7. Enter the step size (typically initialized to the pixel size)

The macro will run extractions at increasing radii from your start point to your end point.

Visualizing Multiple Radius Extractions [00:07:34]

After the macro completes, you'll have multiple 2D images in your workspace. To view them effectively:

  1. Select all the images
  2. Create a box (which has spatial extent over all images)
  3. Select and lock the box
  4. Scroll through the slices to see data at different radii

Alternatively, you can stitch these images into a 3D dataset:
1. Select all the images
2. Right-click and select "Stitch 3D"
3. Choose "Max" for intensity and "Nearest" or "Tri-cubic" for interpolation
4. Click "Stitch"

This creates a single 3D image where you can visualize all the radius extractions together.

Accessing the Extension [00:10:43]

To get this functionality:
1. Go to Utilities > Infinite Toolbox
2. Navigate to "Macros"
3. Find "Extract Soup Can Wrapper" and "Soup Can Wrapper Over Radius Range"
4. Click "Install for all users"

The source code will be available on your system in the menu items folder after installation.

Practical Applications [00:14:09]

This technique is particularly useful for:
- Inspecting cylindrical objects at different depths
- Detecting surface irregularities like divots or protrusions
- Analyzing the internal structure of cylindrical samples

At different radii, you can see how the data changes from the center to the outer surface of your sample.

Conclusion

The cylinder coordinate transform provides a powerful way to visualize and analyze cylindrical data in Dragonfly. While the current implementation is somewhat of a workaround, it offers valuable insights for many scientific and industrial applications. Future improvements may include interactive radius adjustment and better visualization options.

Remember that you can download both the basic extraction tool and the radius range macro from the Infinite Toolbox to enhance your Dragonfly workflow.
