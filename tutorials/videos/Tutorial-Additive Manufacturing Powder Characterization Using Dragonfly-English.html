This video demonstrates how to use watershed segmentation in Dragonfly software to separate touching particles in powder datasets, allowing for individual analysis of particle properties such as volume and sphericity.

Tutorial Content

Watershed Segmentation of Powder Particles Tutorial

Introduction [00:00:02]

This tutorial demonstrates how to perform watershed segmentation of powder particles in a CT scan dataset. Watershed segmentation is particularly useful when you need to separate touching objects in your data to calculate properties for each particle individually.

Dataset Overview [00:00:21]

This demonstration uses a small, cropped powder dataset to simplify the workflow. While we'll start with a slice view, we'll also examine the data in 3D to better understand the particle distribution.

For this type of analysis, it's recommended to use a quad view to see both slice and 3D perspectives simultaneously.

Basic Segmentation [00:01:04]

Before applying the watershed algorithm, we need to create a basic segmentation of the particles:

  1. Navigate to the Segmentation tab
  2. Select Define Range
  3. Choose Upper OTSU - an automated thresholding method to classify voxels as either material or background
  4. Similarly, use Lower OTSU and Add to Network to complete the basic segmentation

At this point, you should have a simple segmentation with particles and background, but touching particles will still be connected.

Creating Seed Points [00:02:11]

The watershed algorithm requires seed points. There are two approaches to create these:

Method 1: Erosion

  • Take the segmented particles and erode them
  • This creates seed points within most particles

Method 2: Distance Map (Recommended)

  1. Select the background
  2. Right-click and select Create Distance Map
  3. This creates values with the highest values being furthest away from your selected region
  4. Apply Upper OTSU to this distance map
  5. Manually adjust the threshold slightly upwards to ensure particles aren't touching while preserving values within particles
  6. This creates a "distance mapped particles" segmentation

Preparing for Watershed [00:03:56]

To prepare the input for the watershed algorithm:

  1. Add the background to the distance mapped particles
  2. Create a union of these by combining them into a new region of interest (name it "seed for watershed")
  3. Turn off other regions to simplify the view, leaving only particles and background

For watershed to work properly, the marking needs to be separated like a multi-ROI, with each separate item having a different color.

Computing the Watershed [00:04:45]

  1. Right-click on the image channel
  2. Select Compute Watershed
  3. This process highlights the edges between objects and calculates the separated objects

The result will be a watershed multi-ROI where each particle has separate values.

Analyzing the Results [00:05:14]

Once the watershed segmentation is complete:

  1. Rename the result for clarity
  2. Calculate particle properties using the Scalar Generator
  3. For demonstration purposes, calculate volume and sphericity
  4. View the result in 3D
  5. Classify particles according to measurements like volume or sphericity
  6. Adjust the color coding as desired (the temperature color map works well)

Conclusion [00:06:38]

This workflow demonstrates how to separate touching particles using watershed segmentation in Dragonfly. While there are other ways to use the watershed segmentation tools, this approach provides a straightforward method to separate particles for individual analysis and property calculation.
