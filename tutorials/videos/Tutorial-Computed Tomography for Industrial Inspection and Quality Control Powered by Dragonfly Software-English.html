This video provides a tutorial on using Dragonfly software for industrial CT scan analysis, demonstrating how to visualize, segment, and quantitatively analyze casting porosity and wall thickness in 3D.

Tutorial Content

Introduction to Industrial CT Inspection with Dragonfly [00:00:02]

This tutorial will help you unlock more information from your CT scans, focusing on industrial inspection. While aimed at new users, the tutorial moves quickly through the basics, so feel free to pause and review as needed.

Getting Started with Dragonfly [00:00:40]

When you first open Dragonfly, you'll see the main interface. Let's start by importing image files:

  1. To load an image stack, select all images in the folder
  2. Click "Add a slice image" and import
  3. Click "Next" and enter the voxel size (in this example, 0.2 millimeters)
  4. Note that the decimal point or comma follows your system's convention

After loading, the data opens as a slice image. To see the quad view, go to the Layout menu and select the quad view option. This allows you to see the 3D render, which can be easily adjusted using the shortcuts.

Basic Navigation and Visualization [00:01:47]

For a quick 3D visualization of your part:

  1. Choose a preset such as Cast Aluminum for a nice initial view
  2. Double-click the view to evaluate the surface easily
  3. Navigate using:
  4. Zoom: Hold the scroll wheel and drag back and forth
  5. Pan: Keep both left and right mouse buttons pressed while dragging
  6. Rotate: Use the translate/rotate tools by clicking on a window to activate it, then turn the object

To align views orthogonally, use the rotation tools and press Escape to exit that mode.

For better visualization:
1. Right-click in the 3D view and add a 2D View Plane function to see your current position
2. Change the background by clicking the background button and selecting Uniform White for clearer viewing

Qualitative Inspection for Defects [00:03:35]

In industrial inspection, you'll want to evaluate cracks and pores qualitatively:

  1. Change the contrast using the Windows Leveling Tool at the top
  2. Choose the opacity gradient option
  3. Drag the left bar to the right and adjust to get good contrast for viewing pores

This gives you a nice view of the largest pores. For castings like this example, you'll see shrinkage porosity inside, which is one of the main applications of CT scanning.

For evaluating porosity across multiple slices:
1. Choose the Minimum View Mode
2. This integrates over multiple slices, showing the lowest value pixel at each location
3. This "thick slab mode" helps identify if pores align along certain orientations

Quantitative Porosity Analysis [00:05:34]

To evaluate porosity quantitatively:

  1. Go to the Segment Tool
  2. Use Define Range for segmentation
  3. You can use automated lower OTSU thresholding
  4. Or set a manual threshold (in this example, 42,000 works well)
  5. This creates a region of interest (ROI) containing all voxels with gray values below the threshold
  6. To remove the outside area, right-click and select Refine the Region of Interest
  7. Choose Process Islands (connected regions)
  8. Remove the largest connected region (which is the outside space)
  9. Untick Define Range to see only the ROI (the porosity)
  10. Change the color if desired (yellow makes it easier to see)

This highlights the largest pores in your region of interest. Smaller pores may be missed due to partial volume effects, which is why quantitative analysis typically focuses on the largest pores.

3D Visualization of Porosity [00:07:20]

To visualize the porosity in 3D:

  1. Change the contrasting to a flat approach with opacity
  2. Activate the 3D view to see the pores
  3. Increase the highlight to make the ROI brighter
  4. Right-click and create a multi-ROI for evaluation

This multi-region of interest is what we use for quantitative analysis. To visualize in 3D:

  1. Untick the region of interest
  2. Tick the 3D view
  3. Select measurements to classify pores according to:
  4. Volume (the largest one appears yellow)
  5. Mean Ferret diameter (the longest diameter)

All this data can be exported to Excel or other formats for further processing.

Wall Thickness Analysis [00:09:58]

To evaluate wall thickness:

  1. Create a new region of interest
  2. Define a range that includes all material (use upper OTSU)
  3. Add to a new ROI
  4. Fill in any pore areas to ensure the ROI covers the entire part
  5. Export to a thickness mesh
  6. Perform smoothing to simplify processing time and visualization

The wall thickness evaluation is important for identifying areas where walls need to be above a certain minimum value. After the analysis completes:

  1. Tick the view button to see the thickness distribution in 3D
  2. Click on the button and move the sample to highlight thickness mesh values
  3. Observe the thickness range (in this example, up to almost 70 millimeters)

This thickness visualization helps identify thin regions that might not be obvious in the original data.

Conclusion [00:13:33]

This tutorial has demonstrated how to use Dragonfly for industrial CT inspection, covering basic navigation, visualization techniques, porosity analysis, and wall thickness evaluation. These tools help extract valuable information from CT data to assess part quality and identify potential issues.

Dragonfly is a powerful tool not only for CT data but also for other imaging modalities, including SEM images and FIB-SEM images.
