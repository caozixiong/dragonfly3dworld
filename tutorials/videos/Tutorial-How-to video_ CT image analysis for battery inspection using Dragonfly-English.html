How-to video: CT image analysis for battery inspection using Dragonfly - (English)

This tutorial demonstrates how to segment battery overhang regions using Segmentation Wizard and deep learning, showing how this approach overcomes the limitations of traditional thresholding methods to achieve more accurate and consistent results.

Tutorial Content

Introduction to Battery Overhang Segmentation [00:00:03]

This tutorial demonstrates how to segment battery overhang regions using Segmentation Wizard, a deep learning-based approach. Rather than showing each step in real-time, this tutorial explains the process and showcases the results on already processed data.

Understanding the Challenge [00:00:42]

The region of interest in this tutorial is the battery overhang. Traditionally, engineers would use linear measurements with a ruler to measure these regions.

When making linear measurements:
- Measurements do not snap to voxels - they are subvoxel measurements
- Text from multiple measurements may overlap
- Use the floating caption feature to organize multiple linear measurements cleanly

While linear measurements work, they are limited. Ideally, we want to segment out the entire overhang regions.

Limitations of Traditional Thresholding [00:02:39]

Traditional thresholding methods have significant limitations when trying to segment battery overhangs:
- Adjusting threshold ranges produces inconsistent results
- Some areas are missed while unrelated areas are captured
- The segmentation appears messy across multiple slices
- Finding an optimal threshold range is extremely difficult

This is precisely why deep learning provides a better solution for this type of segmentation.

Deep Learning Approach with Segmentation Wizard [00:03:21]

Deep learning works well for this application because it uses image recognition techniques. For better results:
- Crop the dataset to focus only on the area of interest
- This helps the model converge to a good solution more easily
- The final result cleanly segments the overhang regions

Applying a Trained Model [00:04:15]

Once you have trained a model, applying it is straightforward:
1. Select your segmentation model (in this case "battery over")
2. Choose a slice to analyze
3. Click on Preview to apply the model to that slice

The key advantages of this approach:
- No human bias or manual thresholding involved
- Simply apply your previously trained model to all slices
- The same model can be used on multiple similar battery scans
- Results can be used for thickness analysis and overhang analysis

Training with Segmentation Wizard [00:05:30]

To access Segmentation Wizard:
1. Navigate to Artificial Intelligence > Segmentation Wizard

The training process involves:
- Creating training frames by painting the regions of interest
- Using the smart grid tool for 2D segmentation
- Fixing areas with the paint tool where needed
- Using a unit attention model for training

You don't need to create all training frames at once:
1. Start with one frame and train initially
2. Add new frames as needed
3. Use your existing model to pre-segment new training frames
4. Fix up the edges or problem areas
5. Retrain to strengthen your model

Before finishing, remember to save your model with a recognizable name so you can apply it later. Training a model is separate from applying it to your entire dataset.

Conclusion [00:08:19]

This deep learning approach to battery overhang segmentation offers significant advantages over traditional methods. By training a custom model with Segmentation Wizard, you can achieve consistent, accurate results without the limitations of threshold-based techniques. The same model can be applied to multiple similar scans, making it an efficient solution for battery analysis workflows.
