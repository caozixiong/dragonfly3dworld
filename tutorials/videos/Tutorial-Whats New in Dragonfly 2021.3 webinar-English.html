Introduction [00:00:00]

Welcome to today's webinar on What's New in Dragonfly 21.3. This presentation will cover several new features in the latest release, focusing primarily on three categories: data organization, visualization, and deep learning. Why Dragonfly 21.3? [00:01:21]

Dragonfly follows a new naming convention where the first release of each calendar year has a .1 appendix. The current release, 21.3, follows the maintenance update 21.2. The release is packed with bug fixes, new features, performance optimizations, and code refactoring that make the software perform better and easier to maintain.

For those interested in exploring all new features, visit objects.com/dragonfly/new, where all release notes are archived. Each release offers something valuable for almost every Dragonfly user.

Data Organization [00:04:15]

Dragonfly Organizer - Redesigned User Interface [00:04:23]

The Dragonfly organizer (or project organizer) has been completely rebuilt with a more functional and user-friendly interface. This tool helps users organize their data and results for more productive Dragonfly sessions.

Key features include:
- Ability to create folders/projects and search through them
- Tagging capabilities to mark projects (e.g., "sandstone", "clastic rock")
- Starred projects list for quick access
- Recent projects list
- Push-pin feature to keep projects visible even after clearing search results
- Storage for ORS objects (images, multi-rois, meshes) and associated files (PDFs, spreadsheets, Word docs)

The organizer is now included for free with every Dragonfly license, whereas it was previously a premium extension. Importantly, project data is not locked in an encrypted database - projects are simply folders that can be copied, browsed with File Explorer, and archived with normal backup utilities.

Intensity Scale Calibration [00:07:54]

CT reconstruction engines often fail to reconstruct the same material at the same grayscale values, making data processing inconsistent. Dragonfly now lets users define their own units (similar to the Hounsfield scale in clinical CT) to calibrate data from different sources.

Benefits of intensity scale calibration:
- Define custom scales for different materials (e.g., setting air to 0 and quartz to 100)
- Create consistent visualization across datasets
- Make image segmentation and data processing more robust
- Improve deep learning model consistency
- Share calibration settings with colleagues

The calibration updates the readout intensity while preserving the raw data values, ensuring data integrity while providing consistent processing.

Visualization [00:12:58]

Transient Planar Views [00:13:09]

When enabled, this feature shows exactly where a view is in 3D space whenever you move a plane or crosshairs. You can also enable a 3D cursor to see precisely where crosshairs would be in 3D. This provides clear visual cues about the position of 2D views within the 3D volume.

Non-Planar Views [00:14:11]

Dragonfly now supports non-planar views for different geometric shapes:

  1. Cylinders: Create a "soup can wrapper" view that unwraps the surface of a cylinder. You can adjust:
  2. Theta offset (which seam appears on the left)

  3. Radius

  4. Spheres: For spherical surfaces (like a ball-and-socket joint), you can:

  5. Fit a sphere to a point set
  6. View the sphere in equirectangular projection

  7. Adjust phi and theta parameters

  8. Planes: For warped or curved planes, you can project them onto a flat plane

All non-planar views maintain correspondence with standard views, allowing you to trace the 3D cursor and perform segmentation directly in these views.

Window Leveling and Lookup Tables [00:17:40]

The window leveling interface has been completely redesigned with:
- Ability to move color nodes and create split colors
- Customizable alpha function curves with control points
- Option to pin window leveling plugin to desktop for working with multiple datasets simultaneously
- Enhanced control over color rendering in both 2D and 3D

The interface now includes:
- Traditional linear ramp
- Left-gated ramp
- Bi-gated ramp
- Bandpass ramp
- New piecewise option for full customization of alpha curves

Enhanced Annotations [00:20:57]

Improvements to on-screen annotations include:
- Rulers with pullout capability in 3D views
- Arrows with drop shadows for better clarity
- Text annotations with customizable background colors and opacity
- Updated scale bar behavior with more varieties to match style needs

Virtual Floor with Reflection [00:22:18]

The virtual floor feature now supports reflection, creating artistic renderings suitable for presentations, publications, or desktop wallpapers.

Deep Learning Enhancements [00:22:51]

Segmentation Wizard Updates [00:23:04]

The Segmentation Wizard has been enhanced with:
- A histogram button to view statistics in each frame (class balance)
- Improved model management with new recommended models
- More accessible and user-friendly data augmentation controls
- Expert controls for fine-tuning models (patch size, stride ratio, loss function)

Visual Feedback [00:25:10]

This powerful new feature in both Segmentation Wizard and the deep learning tool provides visual updates during model training:

  • After each epoch, you see the model's current performance on a preview frame
  • Ability to track progress visually rather than just through loss function numbers
  • Option to scroll through training history to compare earlier states
  • Checkpoints to save the best models (configurable by count or disk space)
  • Ability to restore previous model states if desired

Technical Improvements [00:30:20]

  • Migration from TensorFlow 1.14 to TensorFlow 2.4
  • Support for more custom layers and loss functions
  • Accelerated data augmentation to better utilize GPU processing power
  • Three types of models available:
  • 2D models (single slice)
  • Multi-slice 2D models (2.5D)
  • True 3D models (like UNet3D)
