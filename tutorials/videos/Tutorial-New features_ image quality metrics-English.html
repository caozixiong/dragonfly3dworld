New features: image quality metrics - (English)

This video demonstrates three new features in Dragonfly version 2022.2: the edge contrast visualization tool for better viewing of internal structures, the equivalent spherical diameter calculation for faster analysis of pores or features, and image quality metrics for quantifying differences between original and processed image slices.

Tutorial Content

New Features in Dragonfly Version 2022.2

Introduction [00:00:02]

This tutorial demonstrates three new features in Dragonfly version 2022.2 using a concrete sample that has been micro CT scanned. The sample has been analyzed in previous tutorials to segment and analyze pore spaces.

1. Edge Contrast Visualization [00:00:38]

The first new feature enhances 3D visualization capabilities:

  1. Click on an object in the 3D window
  2. In the 3D settings, locate the new edge contrast tick box and slider
  3. Activate the tick box and adjust the slider to the left

This provides a transparency method that still shows edges of the object on the opposite side and internal features with edges. This visualization technique is particularly effective for:

  • Viewing internal structures
  • Maintaining visibility of edges while using transparency
  • Visualizing porosity with color coding

When the image channel is activated, the pore information color coding legend may not be shown. Click on the porosity multi-ROI to see that the color coding is according to sphericity.

2. Equivalent Spherical Diameter Calculation [00:02:17]

A powerful new calculation feature has been added:

  1. Right-click on the multi-ROI
  2. Go to scalar generator
  3. Under basic measurements, right under volume, find the new "equivalent spherical diameter" option

This calculation determines the diameter of a perfect sphere with the same volume as your pore or feature. The benefits include:

  • Very fast calculation compared to more complex measurements
  • Useful for regular quick analysis
  • Efficient when analyzing thousands of features

For complex analyses with more than 10,000 features, calculating maximum Feret diameter or other complex measurements can be time-consuming. Using equivalent spherical diameter minimizes processing time while providing valuable data.

3. Image Quality Metrics [00:03:11]

The third new feature allows quantitative comparison of image quality between datasets:

Preparing Image Slices for Comparison

  1. Right-click on the image and select "Show marked slices indicator"
  2. Click on a specific slice to mark it
  3. Right-click on the image and select "New image from marked slices"
  4. Repeat the process for the second dataset using the same slice number

Comparing Image Quality

  1. Right-click on either of the single slice images
  2. Select "Open image quality metrics"
  3. Select both single slice images for comparison
  4. Calculate different image quality metrics

The metrics include:
- Sharpness value (significantly lower in blurred/denoised images)
- Shannon entropy
- Regular signal-to-noise ratio
- Q factor calculation

In the example shown, the original data had a sharpness value of 782, while the denoised version had only 137, clearly demonstrating the difference in image quality.

Conclusion [00:06:08]

These three new features in Dragonfly 2022.2 enhance visualization capabilities, provide efficient calculations for large datasets, and enable quantitative image quality comparison. Each feature addresses specific needs in the analysis workflow, making the software more versatile and powerful for micro CT analysis.
