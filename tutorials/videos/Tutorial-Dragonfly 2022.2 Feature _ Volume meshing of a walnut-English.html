Dragonfly 2022.2 Feature : Volume meshing of a walnut - (English)

This tutorial demonstrates how to create a volumetric 3D tetrahedral mesh in Dragonfly version 2022.2, showing the process of converting a segmented multi-ROI into a simulation-ready mesh that can be exported for finite element modeling applications.

Tutorial Content

Creating a Volume Mesh in Dragonfly 2022.2

Introduction [00:00:03]

This tutorial demonstrates how to create a volume mesh in Dragonfly version 2022.2, one of the new features in the latest release. The demonstration uses a small walnut dataset that has already been segmented using watershed segmentation.

Overview of the Data [00:00:30]

For those unfamiliar with Dragonfly, the interface shows a quad view of the data. The dataset is a walnut scan with a base underneath it. Watershed segmentation was necessary because the walnut shell and the base are touching at their interface, making it a challenging segmentation task.

While manual segmentation is possible by creating a multi-ROI with different classes (such as "walnut" and "everything else") and painting slice by slice, this tutorial focuses on the volume mesh creation rather than segmentation techniques.

Purpose of Volume Meshing [00:01:40]

A volumetric or 3D tetrahedral mesh can be used for simulation purposes. For example, you might want to export the walnut model to a finite element modeling code to simulate how much force it would take to crack the walnut.

Creating a Volume Mesh [00:02:01]

Prerequisites

To create a volume mesh in Dragonfly, you need:
- A multi-ROI with two or more classes (this demo uses "walnut" and "everything else")

Steps to Generate the Mesh

  1. Highlight the regions by pressing the Control button and clicking with the left mouse button to activate the volume
  2. Activate both classes in the multi-ROI
  3. Right-click and select Create a 3D tetrahedral mesh

Optimizing Processing Time [00:02:40]

Creating a tetrahedral mesh requires significant CPU processing. To make calculations faster, especially when starting out:

  1. Create a smaller voxel dataset by downsampling
  2. You can downsample either the image data or just the region of interest (multi-ROI)
  3. Right-click on the multi-ROI
  4. Go to Modify → Transform → Sample
  5. Set a downsampling factor (in this demo, a factor of 10 is used to make a very short calculation)

Generating the Mesh [00:03:38]

  1. Right-click on the downsampled version
  2. Select 3D tetrahedral mesh generator
  3. Use the default settings or adjust as needed
  4. Click Generate mesh

The process takes only a few seconds with the downsampled data.

Viewing the Mesh [00:04:14]

After generation, a new object appears in the project panel:

  1. Click on the 3D tetrahedral mesh
  2. Enable the Scalar information tick box to create color on the mesh
  3. This displays the different regions of the multi-ROI with mesh elements fitting together

Creating a Mesh for a Single Region [00:04:47]

If you only want to mesh one region (e.g., just the walnut, not everything around it):

  1. Click on just the walnut in the multi-ROI
  2. Right-click and select 3D tetrahedral mesh generator
  3. Click Generate mesh

This creates a full volumetric mesh representing not just the surface (like an STL surface mesh) but a complete volumetric mesh suitable for simulation.

Exporting the Mesh [00:05:29]

To save the mesh file:

  1. Right-click on the tetrahedral mesh
  2. Select Export tetrahedral mesh to file
  3. Choose a format:
  4. Netgen mesh (from NGSolve) is the default format
  5. Other formats are also available for direct export

The mesh can be easily converted to other formats as needed.

Conclusion

This tutorial demonstrated how to create and export a volumetric mesh in Dragonfly 2022.2. The resulting tetrahedral mesh can be used for various simulation purposes in external applications. The process involves creating a multi-ROI, downsampling for efficiency, generating the mesh, and exporting it in the desired format.
