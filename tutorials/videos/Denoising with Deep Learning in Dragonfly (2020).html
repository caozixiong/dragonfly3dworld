Dragonfly Daily 16: Denoising with Deep Learning in Dragonfly (2020)

This video, presented by Dragonfly Product Manager Mike Marsh, introduces the use of deep learning methods for denoising scientific images.

Introduction to Deep Learning Denoising

This session focuses on the application of deep learning techniques for scientific image denoising. Deep learning requires an NVIDIA CUDA-enabled graphics card. The presenter utilizes a high-performance workstation for deep learning tasks.

Neural Network Model Construction

The core concept involves training a neural network to transform noisy input images into low-noise output images. The training paradigm involves pairing inputs with outputs, specifically pairing "cheap" (high-noise) images with "expensive" (low-noise) images.

Three Deep Learning Denoising Methods

There are three deep learning denoising methods:

  • High-Noise to Low-Noise: Pairing high-noise images with corresponding low-noise images.
  • Autoencoder (High-Noise to High-Noise): Training the network to reconstruct high-noise images from themselves.
  • Noise to Noise: Training the network to transform one noisy image into another independent noisy version.

Practical Demonstration in Dragonfly

The video demonstrates how to set up and train a denoising model using deep learning tools within Dragonfly. It also briefly covers the concept of transferable models and the Infinite Toolbox.

Comparison of Denoising Results

The video compares deep learning-based denoising results with traditional image filtering methods. Deep learning can preserve more texture and features than some smoothing filters.

Additional Information

The presenter announces an upcoming workshop on image processing techniques. A Q&A session is also held, discussing various questions related to denoising, deep learning, and Dragonfly.
