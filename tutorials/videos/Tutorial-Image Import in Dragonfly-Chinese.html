Dragonfly 3: 图像导入功能教程

介绍 [00:00:00]

这个Dragonfly 3培训视频涵盖了图像导入的功能。我们将探索三个不同的例子：

  1. 导入二维图像堆栈（例如，TIFF文件）
  2. 导入三维体积文件（例如，3D TIFF或.raw/.dat组合）
  3. 导入元数据未编码在文件中的.raw文件

导入TIFF文件堆栈 [00:00:31]

启动Dragonfly时，通常会看到导入数据集对话框。在这里，您可以：

  • 选择包含TIFF文件的文件夹，或
  • 点击add按钮并导航到包含TIFF文件的文件夹以选择特定文件

要选择多个文件：
1. 选择第一个TIFF文件
2. 向下滚动并按住shift键点击选择最后一个TIFF文件
3. 点击open

您选择的图像将出现在队列中。您可以：
- 点击next继续
- 选择任何单个图像并点击see preview查看快照
- 通过点击预览图像来更改预览切片

导入参数配置

导入数据集对话框的第二页允许您配置各种参数：

  • 数据集名称：为您的数据集输入自定义名称
  • 像素大小：如果Dragonfly从文件元数据中检测到像素大小，它将被显示。如有需要，您可以修改这些值。
  • 降采样：通过在x、y和z维度上降采样来减少数据大小

裁剪图像

减少数据占用空间的另一种方法是裁剪图像：

  1. 点击crop image查看第一个切片的预览
  2. 滚动浏览数据以识别感兴趣的区域
  3. 设置x、y和z维度的边界
  4. 点击OK应用裁剪

这允许您在不降采样的情况下消除边缘上的多余数据，更有效地利用内存。

其他导入选项

您还可以：
- 编码数据偏移和斜率（很少需要）
- 应用物理转换系数进行线性变换
- 沿x、y或z轴应用镜像变换
- 在导入时交换轴或执行轴变换

导入没有元数据的3D Raw文件 [00:05:09]

对于元数据未编码的raw文件：

  1. 选择raw文件并点击next
  2. 手动输入矩阵维度（例如，1024×1024×436）
  3. 选择适当的位深度（8位、16位、32位整数或32位浮点）
  4. 通过勾选相应的框来指定数据是否为有符号（非无符号）
  5. 对于大端格式的数据，勾选反转字节顺序复选框

一旦参数正确输入，图像预览应该看起来合理。然后您可以：

  1. 点击next进入第二页
  2. 为数据集输入名称
  3. 指定x、y和z维度的像素大小
  4. 点击finish导入数据集

导入3D体积文件 [00:07:24]

当导入3D体积文件（如3D TIFF或.raw/.dat组合）时：

  1. 选择.dat文件（包含元数据）
  2. 点击next
  3. Dragonfly将自动解码三维信息

导入具有相同几何形状的多个数据集

如果您需要导入具有相同几何形状的多个数据集：

  1. 使用裁剪功能定义感兴趣的区域
  2. 勾选keep same geometry选项
  3. 点击continue
  4. 选择第二个数据文件
  5. 相同的裁剪参数将自动应用
  6. 点击finish加载具有相同占用空间的两个数据集

然后您可以为每个数据集分配不同的颜色以进行可视化。

结论 [00:09:22]

Dragonfly中的导入数据集功能允许您：
- 导入图像文件作为TIFF堆栈
- 导入有或没有编码元数据的3D体积
- 在导入过程中更改像素大小、重命名数据集和应用变换
- 裁剪数据并导入具有相同几何形状的多个通道

这些功能为将各种类型的图像数据导入Dragonfly提供了全面的解决方案。
