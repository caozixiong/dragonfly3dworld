Dragonfly Daily 36: CT Reconstruction in Dragonfly 2020.1

A step-by-step guide to creating stunning 3D reconstructions from your CT data

Introduction

This tutorial will guide you through the process of performing CT reconstruction using Dragonfly 2020.1. We'll cover the key steps involved, from loading your projection data to visualizing the reconstructed volume.

Prerequisites

  • Dragonfly 2020.1 software
  • CT projection data

Steps

  1. Load Projection Data
    • Open Dragonfly 2020.1.
    • Go to the File menu and select Import Data.
    • Choose the folder containing your projection data.
    • Select the projection data files (typically TIFF or HDF5 format).
    • Click Open.
  2. Access the CT Reconstruction Workflow
    • Go to the Workflows menu and select CT Reconstruction (Beta).
    • The CT Reconstruction dialog will appear.
  3. Select Input Image
    • In the Input Image dropdown menu, select the projection dataset you want to reconstruct.
  4. Choose Reconstruction Engine
    • Select the type of reconstruction engine you want to use:
      • RTK: For cone beam reconstruction.
      • Parallel Beam Reconstruction: For parallel ray reconstruction.
  5. Specify Geometry Parameters
    • Enter the geometric parameters for your CT setup:
      • Source to detector distance
      • Source to ISO center distance
      • Initial angle of rotation
      • Angular step
      • Source offset
      • Detector angle
  6. Import Parameters (Optional)
    • If you have a parameter file associated with your CT system (e.g., from Nikon), you can import it by clicking the Import Parameters button.
  7. Select Reconstruction Algorithm
    • For cone beam reconstruction, the FDK algorithm is used.
    • For parallel beam reconstruction, you can choose between Gridrec or ART.
  8. Configure Algorithm Parameters
    • Depending on the chosen algorithm, you can configure additional parameters such as:
      • Compute on GPU
      • Number of iterations
      • Forward projected models
  9. Apply Filters (Optional)
    • You can apply filters to improve the quality of the reconstruction. Some common filters include Shepp-Logan, Hann, and Hamming.
  10. Pre-processing (Optional)
    • You can perform pre-processing steps such as flat field correction or median filtering to improve the projection data.
  11. Compute Preview (Optional)
    • You can compute a preview of the reconstructed volume to check the results before performing the full reconstruction.
  12. Reconstruct and Load or Reconstruct and Save
    • Choose whether to reconstruct the entire volume and load it into Dragonfly (if you have sufficient memory) or reconstruct and save it to disk.

Conclusion

By following these steps, you can perform CT reconstruction in Dragonfly 2020.1 and visualize your data in 3D.
