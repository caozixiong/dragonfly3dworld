Dragonfly Daily 02: Working with Meshes in Dragonfly (2020) - Summary and Tutorial

Video Overview

This video, presented by Mike Marsh, Product Manager for Dragonfly, serves as Lesson 2 in a series focused on the Dragonfly scientific imaging application. It provides an introduction to working with meshes within the Dragonfly environment. The lesson covers mesh rendering, generation, importing, visualization, and saving.

Step-by-Step Tutorial: Working with Meshes in Dragonfly

  1. Downloading a Sample Mesh Data Set [00:02:26]

    • Navigate to the MorphoSource repository to acquire scientific imaging and mesh data.
    • Search for the identifier "MorphoSource 104845" to locate the specific mandible data set used in the video.
    • Create an account and log in to MorphoSource to download the data set, which is provided under a Creative Commons license by the National Museum of Natural History.
    • The downloaded file will be in a ZIP archive; extract the contents to access the .PLY mesh file.

  2. Importing and Visualizing a Mesh File [00:04:33]

    • Open Dragonfly and navigate to File > Import Mesh Files.
    • Browse to the location of the downloaded .PLY file and import it.
    • Note that the initial display shows the mesh as an outline.
    • To view the mesh in 3D, adjust the layout to a 3D or multi-view configuration.
    • Explore the Mesh Property panels to toggle between solid, outlined, and wireframe renderings [00:06:08].

  3. Generating a Contour Mesh from Image Data [00:07:09]

    • Load an image data set into Dragonfly (e.g., "fracture in Granite").
    • Right-click on the image data set and select Generate Contour Mesh [00:07:49].
    • Accept the default settings and export the contour as a mesh.
    • The resulting mesh represents the interface within the image data.
    • Adjust the mesh's color and transparency via the color tile in the properties panel [00:08:49].
    • Apply smoothing to the mesh to refine its surface [00:09:00].

  4. Importing Additional Mesh Files [00:09:29]

    • Download a mesh file from a public repository such as Thingiverse (e.g., "floppy dragonfly").
    • Import the downloaded .STL file into Dragonfly using File > Import Mesh Files [00:10:15].
    • Customize the appearance of the imported mesh by adjusting color and transparency.
    • Utilize the 3D visualization tools within Dragonfly to manipulate the view, lighting, and shadowing.
    • Experiment with "eye candy" features like depth of field for enhanced rendering [00:11:03].

  5. Visualizing Data on a Mesh [00:12:06]

    • Import a mesh file that contains encoded data (available from the ORS Dragonfly sample data sets).
    • Right-click on the mesh and select Measurement Inspector [00:12:41].
    • Choose a measurement to visualize (e.g., volume or surface area).
    • Adjust the color map and range to effectively display the data distribution on the mesh.
    • Use transparency settings to highlight specific data ranges [00:13:35].

  6. Saving a Mesh [00:14:34]

    • To save a mesh, right-click on it and choose Export as ORS Object.
    • This will preserve all associated data, such as measurements encoded on the vertices.

  • Considerations for slice distance in FIB-SEM data [00:17:44].

Conclusion

This lesson provides a foundational understanding of working with meshes in Dragonfly, covering essential techniques for importing, visualizing, generating, and saving mesh data. It also addresses common questions and considerations related to mesh processing and analysis.
