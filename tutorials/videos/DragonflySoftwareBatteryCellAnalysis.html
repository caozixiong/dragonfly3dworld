Dragonfly Software for Battery Cell Analysis

This tutorial demonstrates how to analyze battery cells using Dragonfly software, covering manual measurements, defect inspection, and automated analysis with deep learning.

I. Initial Data Visualization and Orientation

  • Data Loading and Layout: Load battery cell data in Dragonfly 2024.1. If the data isn't displayed in quad view, change the layout in the layout panel [00:00:37].
  • View Manipulation: Maximize or minimize specific views by double-clicking. Use the flip tool to orient the cell upright [00:01:16].
  • Clipping Box: Use the clipping box to virtually cut open the data and inspect internal structures like anodes, cathodes, and the metal casing [00:02:11].

II. Manual Inspection and Annotation

  • Cross-sectional Views: Double-click on 2D cross-sectional views to make measurements. Identify anodes, cathodes, and spacer materials [00:03:08].
  • Oblique Views: Use the mouse near crosshairs to create an oblique view, allowing for circumferential inspection [00:04:36].
  • Annotation Points: Use the "Points" tool (with Ctrl+click) to mark defects. Export these annotations with their XYZ locations to a CSV file [00:06:01].

III. Manual Measurements

  • Ruler Tool: Use the ruler tool in the annotation panel to measure overhang heights [00:09:54].
  • Adjustments and Visualization: Adjust measurements by zooming in and using the nearest interpolation mode to see individual pixels [00:10:21].
  • Floating Captions: Use floating captions to position measurements clearly. Export screenshots and ruler data (lengths, XYZ locations) to CSV files [00:11:11].

IV. Automated Analysis with Deep Learning

  • Deep Learning Model Application: Use a pre-trained deep learning model to segment overhang regions adjacent to cathodes [00:13:55].
  • Multi-ROI Analysis: Export the segmentation as an ROI, then as a multi-ROI to compute measurements such as 2D major axis length [00:15:08].

V. Training a Deep Learning Model

  • Segmentation Wizard: Use the segmentation wizard to train a custom model. Define classes (e.g., cathode, background, anode overhang) [00:17:05].
  • Frame Definition: Add multiple frames, marking different regions of the sample. Use a "mixed" frame for training and validation, and another for monitoring [00:17:41].
  • Training Process: Start training, selecting a pre-trained model for faster results. Monitor the loss values and preview the segmentation [00:21:42].
  • Interactive Training: Add more data or adjust existing annotations if the model doesn't converge as expected [00:25:35].
  • Saving the Model: Save the trained model via the model tab or upon exiting the wizard, making it available for use in Dragonfly [00:36:31].
