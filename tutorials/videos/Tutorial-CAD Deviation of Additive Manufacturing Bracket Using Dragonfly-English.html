CAD Deviation of Additive Manufacturing Bracket Using Dragonfly - (English)

This tutorial demonstrates how to align a mesh to CT scan data and perform a CAD deviation analysis in Dragonfly software, showing the process of creating a contour mesh, aligning it with the design file, and generating a color-coded deviation map to visualize differences between the manufactured part and its design.

Tutorial Content

Introduction to Mesh Alignment and CAD Deviation Analysis [00:00:03]

This tutorial demonstrates how to align a mesh to a CT data set and perform a CAD deviation analysis (part-to-design comparison) in Dragonfly. We'll align a bracket with its design file to analyze how well the actual part compares with the design specifications.

Understanding the Initial Data [00:00:29]

The starting point includes:
- CT image data of a bracket (visible in quad view)
- A design file (STL mesh) that is not aligned with the CT data
- The design file is named "stub axle support"

For deviation analysis, we need to work with meshes - both the design mesh and a mesh created from the CT scan.

Creating a Mesh from the CT Data [00:01:23]

To create a mesh from the CT scan:

  1. Go to the segmentation tab
  2. Select Define Range Upper OTSU for automatic thresholding
  3. This finds the optimal threshold value for defining the material (31473 in this example)
  4. Right-click on the image channel and select Generate Contour Mesh
  5. Use a threshold value close to the OTSU value (31450)
  6. Optionally downsample to create a smaller mesh

The result is a contour mesh representing the surface of the scanned bracket. The white line visible in the image view represents the sub-voxel determined surface.

Aligning the Meshes [00:03:13]

To align the contour mesh with the design mesh:

  1. First, perform a basic alignment:
  2. Select Align Centroid to bring the objects closer together

  3. This aligns the centers of the objects as a first step

  4. Perform manual alignment:

  5. Use the Translate Rotate tool on the contour mesh
  6. Manually adjust the position to get a rough alignment

  7. Press the Escape key to exit this mode when finished

  8. Perform automatic registration:

  9. Use the mesh registration tool to align the contour mesh to the design
  10. Apply the default parameters
  11. This completes the fine alignment automatically

If needed, you can also align the original image data to match the aligned mesh:
- Right-click on the image
- Select Modify > Transform > Apply Transformation From
- Choose the contour mesh transformation

Creating a Deviation Map [00:05:54]

Once the meshes are aligned, create a deviation map to visualize differences:

  1. Select the design mesh (stub axle support)
  2. Apply a mesh deviation map

  3. This compares the two meshes to identify deviations

  4. Adjust the visualization:

  5. In the 3D view, change the design mesh to wireframe display
  6. Adjust the color and thickness for better visibility
  7. This makes it easier to see where the actual part deviates from the design

Customizing the Deviation Display [00:07:04]

To improve the visualization of deviations:

  1. Click on the color mapping

  2. Change to a temperature color map for better contrast

  3. Adjust the measurement range:

  4. Right-click and select Measurement Inspector
  5. Set a custom range (e.g., -0.5mm to 0.5mm)
  6. Ensure "Hide out of range measurements" is unchecked to see all deviations

With these settings, you can easily identify:
- Blue areas: where the actual part extends beyond the design
- Red areas: where the actual part is smaller than the design

You can also flip the colors if needed for better visualization.

Conclusion [00:08:14]

This tutorial has demonstrated how to:
- Create a mesh from CT scan data
- Align a scanned mesh with a design mesh
- Generate a color-coded deviation map
- Customize the visualization for better analysis

This process allows for detailed comparison between manufactured parts and their design specifications, helping to identify manufacturing issues or deviations.
