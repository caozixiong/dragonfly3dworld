Dragonfly 3: Image Import Features Tutorial

Introduction [00:00:00]

This Dragonfly 3 training video covers the features of image import. We'll explore three different examples:

  1. Importing a stack of two-dimensional images (e.g., TIFF files)
  2. Importing a three-dimensional volume file (e.g., 3D TIFF or .raw/.dat combination)
  3. Importing a .raw file where the metadata is not encoded in the file

Importing a Stack of TIFF Files [00:00:31]

When launching Dragonfly, you'll typically see the import datasets dialog. From here, you can:

  • Choose to select a folder that contains TIFF files, or
  • Click the add button and navigate to the folder with TIFF files to select specific files

To select multiple files:
1. Choose the first TIFF file
2. Scroll down and shift-click to choose the last TIFF file
3. Click open

Your selected images will appear in the queue. You can:
- Click next to proceed
- Select any individual image and click see preview to view a snapshot
- Change the preview slice by clicking on the preview image

Import Parameters Configuration

The second page of the import datasets dialog allows you to configure various parameters:

  • Dataset Name: Enter a custom name for your dataset
  • Pixel Size: If Dragonfly detects the pixel size from the file metadata, it will be displayed. You can modify these values if needed.
  • Downsampling: Reduce the data size by downsampling in x, y, and z dimensions

Cropping the Image

Another way to reduce the data footprint is to crop the image:

  1. Click crop image to see a preview of the first slice
  2. Scroll through the data to identify the region of interest
  3. Set boundaries for x, y, and z dimensions
  4. Click OK to apply the crop

This allows you to eliminate superfluous data on the margins without downsampling, working more efficiently with your memory.

Additional Import Options

You can also:
- Encode data offset and slope (rarely needed)
- Apply physical conversion coefficients for linear transforms
- Apply mirror transforms along x, y, or z axes
- Swap axes or perform axis transforms upon import

Importing a 3D Raw File Without Metadata [00:05:09]

For raw files where metadata is not encoded:

  1. Select the raw file and click next
  2. Enter the matrix dimensions manually (e.g., 1024×1024×436)
  3. Select the appropriate bit depth (8-bit, 16-bit, 32-bit integers, or 32-bit floating point)
  4. Specify if the data is signed (not unsigned) by checking the appropriate box
  5. For data in Big Endian format, check the reverse byte ordering checkbox

Once the parameters are correctly entered, the image preview should look sensible. You can then:

  1. Click next to proceed to the second page
  2. Enter a name for the dataset
  3. Specify the pixel size in x, y, and z dimensions
  4. Click finish to import the dataset

Importing a 3D Volume File [00:07:24]

When importing a 3D volume file (like a 3D TIFF or .raw/.dat combination):

  1. Select the .dat file (which contains the metadata)
  2. Click next
  3. Dragonfly will automatically decode the three-dimensional information

Importing Multiple Datasets with Same Geometry

If you need to import multiple datasets with identical geometry:

  1. Use the crop feature to define the region of interest
  2. Check the keep same geometry option
  3. Click continue
  4. Select the second data file
  5. The same crop parameters will be applied automatically
  6. Click finish to load both datasets with the same footprint

You can then assign different colors to each dataset for visualization.

Conclusion [00:09:22]

The import datasets feature in Dragonfly allows you to:
- Import image files as a stack of TIFFs
- Import 3D volumes with or without encoded metadata
- Change pixel size, rename datasets, and apply transforms during import
- Crop data and import multiple channels with the same geometry

These capabilities provide a comprehensive solution for importing various types of image data into Dragonfly.
