Non-Planar View for Dewarping tutorial - (English)

This video demonstrates how to use visual RBF rectangles to correct warped or distorted planes in X-ray laminography images of electronics, allowing for sharper visualization and more accurate segmentation of features like copper traces on PC boards.

Tutorial Content

Introduction to X-ray Laminography Visualization [00:00:05]

This tutorial demonstrates how to work with X-ray laminography images of a small PC board with electronics packaging. When rotating and examining planar slices of such samples, it's common to encounter issues where the image isn't sharp across the full width of the plane. This occurs because while the virtual plane is perfectly planar, the actual sample may be distorted, warped, or bent.

Creating Non-Planar Views with Visual RBF Rectangles [00:00:36]

To address warping issues, a non-planar view solution can be implemented using the following steps:

  1. Choose any two-dimensional view of your sample
  2. Select the annotate rectangle tool
  3. Define an initial rectangle on a 2D plane
  4. Right-click and select "retouch"

The rectangle will convert to a radial basis surface function (RBF), which creates a non-planar, distortable plane for better visualization.

Adjusting the Non-Planar View [00:01:17]

To optimize the non-planar view:

  1. Select the desired view (such as the lower left-hand view)
  2. Go to view mode and select "non-planar view"
  3. If needed, apply a horizontal flip to synchronize the orientation

After these steps, the view will display with control points that can be adjusted to improve image quality:

  • Nine control points appear on the screen initially
  • Hold down the control key and drag the mouse up or down to position any point
  • This works like adjusting a trampoline - pushing or pulling to increase the sharpness of the plane being inspected
  • Make adjustments in areas where the image isn't sharp
  • Continue adjusting until you achieve a much sharper view than the original

Adding More Control Points [00:02:27]

To gain finer control over the non-planar adjustments:

  1. Right-click on the visual RBF rectangle
  2. Select "resample" or "redistribute the control points"
  3. Choose a larger grid (e.g., five by five) to add additional control points
  4. Use these additional points to push or pull and correct distortions with greater precision

After making adjustments, fine-tune the brightness and contrast to optimize visualization of features that may be slightly out of plane due to warping or bending in the imaged object.

Moving the Visual RBF Rectangle [00:03:06]

The visual RBF rectangle can be repositioned to work with different planes:

  1. Select the move tool
  2. Drag the rectangle up or down to move through the dataset
  3. Position it on a different plane that may have different warping characteristics
  4. Adjust the control points to optimize the dewarping on the new plane
  5. Add more control points if needed for finer control

Exporting and Working with Non-Planar Views [00:03:58]

After creating an optimized non-planar view, you can:

  1. Export the resampled plane as a separate 2D image
  2. Right-click on the visual RBF and select "extract projection of a structured grid"
  3. Choose to export either the image or its segmentation

Segmentation in Non-Planar Space [00:04:22]

You can perform segmentation directly in the non-planar view:

  1. Create a region of interest (ROI) in your CT scan
  2. Name the ROI (e.g., "copper trace") and assign a color
  3. Use the ROI painter tool
  4. Select an appropriate segmentation method (e.g., multi-slice local OTSU)
  5. Begin painting in the non-planar view

When painting in the non-planar view, the paint will appear on the cross-section because it maps back to the original three-dimensional Cartesian coordinate system. This provides the flexibility to distort the view and then paint or edit in that distorted plane.

Conclusion [00:05:05]

The visual RBF rectangle feature provides a powerful way to correct for warping and distortion in X-ray laminography images. By creating non-planar views and adjusting them with control points, users can achieve sharper visualizations of features that would otherwise be difficult to see clearly. This technique is particularly valuable for applications requiring precise inspection or segmentation of warped samples.
