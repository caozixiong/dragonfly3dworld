July 2022 Workshop Video 6: vector fields, graphs and Porous Microstructure Analysis (PuMA) - (English)

This video demonstrates advanced visualization techniques in Dragonfly software, focusing on vector fields for analyzing bone structure directionality and the PUMA (Porous Microstructure Analysis) plugin for simulating fluid flow and generating visual representations of porous materials.

Tutorial Content

Introduction to Vector Fields and PUMA Plugin [00:00:03]

This tutorial demonstrates several key tools in Dragonfly, including vector fields and the Porous Net Microstructure Analysis (PUMA) plugin. We'll explore these tools using human trabecular bone data and artificially generated porous materials.

Local Thickness Mapping on Trabecular Bone [00:01:02]

Local thickness mapping is particularly useful for bone analysis. This builds on concepts from previous lessons but with a more interesting application.

  1. Import a pre-segmented region of interest from human femur bone
  2. Create a sub-sampled and smoothed thickness map of the trabecular bone
  3. Visualize the thickness map by clicking the visualization button

To properly view the thickness map in 3D:
- Go to Layout Views > Quad View
- Click on the 3D window to see the bone structure visualization

Customizing the Thickness Map Display

The thickness map can be customized in two ways:
- Click directly on the lookup table to change the color scheme
- Right-click and select Measurement Inspector to modify the range

In the Measurement Inspector:
- Adjust the range values to focus on specific thickness values
- Try changing from 1×10⁻⁵ up to 5×10⁻⁵ to better visualize the distribution

For better visualization:
- Try different lookup tables (e.g., "Temperature")
- Use the flip button to reverse the color table (making thinnest parts red and thickest parts blue)

Vector Field Calculation and Visualization [00:04:56]

Vector fields help visualize the directionality of bone structures.

To create a vector field from anisotropy:
1. Select the region of interest
2. Go to Create Mapping > Compute Anisotropy
3. Select "Create an anisotropy vector field"
4. Set appropriate spacing (smaller spacing for more detail)

Once calculated, the vector field appears as arrows showing directionality:
- Adjust arrow size for better visibility
- Make the region of interest mesh more transparent to see vectors clearly
- Change color mapping to highlight strongest vectors (red) and weakest vectors (blue)

The vector field shows how directionality changes smoothly across the region of interest, with the strongest directionality appearing in red sections.

Creating Artificial Porous Materials [00:07:34]

The Poor Network Modeling plugin can generate artificial porous materials for testing and simulation.

To create a random porous material:
1. Go to Workflows > Poor Network Modeling > Generators
2. Set dimensions (e.g., 30×30×30 voxels)
3. Set porosity (e.g., 60%)
4. Click Generate

PUMA Plugin for Microstructure Analysis [00:08:45]

The PUMA (Porous Microstructure Analysis) plugin from NASA allows for various simulations on porous materials:
- Permeability
- Tortuosity (calculation of path distance through an object)
- Thermal conductivity
- Orientation detection

To run a permeability simulation:
1. Select the generated porous material
2. Choose simulation direction (e.g., Y-axis for top to bottom)
3. Select to simulate velocity vectors
4. Compute the simulation
5. Click "Publish" to make results available

Creating Vector Fields from Simulation Results

After running the simulation:
1. Highlight all channels (X, Y, Z, and modulus)
2. Right-click and select "Generate vector field from data sets"
3. Verify the vector components are correctly assigned
4. Create the vector field

To visualize the vector field:
- Go to the main tab and view the vector field in 3D
- Adjust arrow size for better visibility
- Change the color map to highlight maximum flow velocity (yellow) and minimum velocity (blue)
- Optionally select to view only the largest velocities
- Make the material transparent to see where maximum flow occurs

Note: These calculations are memory-intensive. Start with small datasets before moving to large-scale data.

Creating Graph Representations of Pore Space [00:12:45]

Graph representations can be useful for visualizing pore networks.

To create a graph representation:
1. Segment the pore space using appropriate thresholding (e.g., OTSU)
2. Go to 3D Modeling
3. Choose between sparse graph or dense graph:
- Sparse graph: Places balls in the biggest pore spaces connected by segments
- Dense graph: Connects many balls along curved paths

These graph representations can be useful for visualizing pore networks, and scalar values from simulations can be mapped onto them.

Conclusion

This tutorial covered several advanced visualization and analysis techniques in Dragonfly:
- Local thickness mapping for bone analysis
- Vector field calculation and visualization
- Creating artificial porous materials
- Running simulations with the PUMA plugin
- Creating graph representations of pore networks

These tools provide powerful ways to analyze and visualize complex 3D structures, particularly for porous materials like trabecular bone.
