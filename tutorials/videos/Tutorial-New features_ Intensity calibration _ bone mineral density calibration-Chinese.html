New features: Intensity calibration / bone mineral density calibration - (Chinese)

本教程演示如何在Dragonfly软件中进行强度校准以测量骨矿物质密度，展示如何使用已知密度的参考模体将标准灰度值转换为校准后的密度值。

Tutorial Content

Dragonfly中的强度校准

介绍 [00:00:03]

本教程演示如何在Dragonfly中执行强度校准。这种技术特别适用于CT数据中灰度的骨矿物质密度校准，但也可应用于其他目的。演示使用了蜥蜴骨质鳞片、蜥蜴骨骼和骨矿物质密度校准模体的CT数据。

这些数据来自《解剖学杂志》上发表的一项研究，该研究调查了骨质鳞片密度，以确定蜥蜴体内这些骨板是否可以作为钙储存库。

理解校准需求 [00:00:57]

在16位CT数据中，灰度值通常范围从0到65,535。强度校准的目标是将这些任意灰度值转换为有意义的密度测量值。在本例中，我们知道骨矿物质密度校准模体的密度为1200毫克/立方厘米，这将作为我们的参考点。

创建感兴趣区域 [00:01:43]

要校准强度刻度，首先创建校准模体的分割：

  1. 使用局部Otsu刷工具和3D球形画笔选择模体内的高密度像素
  2. 确保选择足够的像素（本例中约1000万体素），同时不包括其他结构
  3. 检查直方图以确定模体的平均灰度值
  4. 在本例中，值集中在约45,292（大约45,000）

校准强度刻度 [00:02:57]

一旦确定了参考材料的灰度值：

  1. 右键点击图像通道
  2. 在上下文菜单底部选择"校准强度刻度"
  3. 选择适当的校准单位（骨矿物质密度预装在Dragonfly中）
  4. 设置参考点：
  5. 双击1200值（代表1200 mg/cm³）
  6. 输入您测量的灰度值（本例中为45,000）
  7. 点击"校准"并关闭对话框

调整显示 [00:04:37]

校准后，图像最初可能显示为空白，因为：

  1. 显示仍设置为显示原始范围（最高65,000）
  2. 校准后的值现在集中在1,000左右

要解决此问题：
1. 转到窗位调整工具
2. 点击"重置"
3. 显示将调整为显示约-400到1,900的值

验证校准 [00:04:55]

要验证校准是否正确：

  1. 在骨质鳞片和校准模体上绘制标尺注释
  2. 检查剖面强度
  3. 确认校准模体显示预期的平均值1,200
  4. 观察骨质鳞片的值高于1,200，并在其宽度上逐渐下降

结论 [00:05:27]

Dragonfly中的强度校准允许您将任意灰度值转换为有意义的密度测量值，这对骨矿物质密度研究特别有用。这种技术通过提供标准化的密度值而非相对灰度强度，实现了CT数据的定量分析。
