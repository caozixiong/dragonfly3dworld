Cylinder Coordinate Transform in Dragonfly (2020)

This video, titled "Cylinder Coordinate Transform in Dragonfly (2020)," focuses on extracting data from a cylindrical perspective using Dragonfly 2020.1 release candidate 2. Here's a breakdown of the video:

Introduction

The host, Mike Marsh, introduces the topic of cylinder coordinate transforms and refers viewers to the ORS YouTube playlist and encourages them to subscribe.

Data Set

A dry sand pack data set from digital rocks portal is used, and the host thanks the scientific community for sharing data.

Cylinder Perspective

The video explains how to inspect data from a cylindrical perspective, imagining a "soup can unwrapper."

Extension

An extension from the infinite toolbox is used to create a cylinder and extract data.

Macro

A macro is used to repeat the analysis at different radii.

Demonstration

The video demonstrates how to use the macro to extract and visualize data at different radii.

Code

The code used for the macro and menu item is discussed, and viewers are shown where to find it in the infinite toolbox.

Source Code

The source code from Nikola is shown.

Final Result

The final result of unwrapping the cylinder over a range of radii is displayed.

Questions and Answers

The host answers questions about SEM processing, macro compatibility, and future improvements to the tool.

Future Webinars

Upcoming webinars on application topics, programming extensions, and image processing are announced.
