This instructional video demonstrates how to create 3D visualization videos in Dragonfly software, showcasing techniques for clipping, segmentation, and animation using a three-horned chameleon specimen as an example.

Tutorial Content

Creating Videos in Dragonfly

Introduction [00:00:04]

This tutorial focuses on creating videos in Dragonfly, demonstrating both 2D slice videos and 3D visualization videos. The demonstration uses a three-horned chameleon dataset.

Setting Up Render Settings [00:00:31]

Before creating videos, it's important to establish appropriate render settings:

  • Select the 3D window to make the 3D presets and settings visible on the right panel
  • Use the main control to select the rendering style first
  • Refine using the histogram on the left
  • Choose a solid rendering style for better visualization

Creating 2D Slice Videos [00:01:54]

2D slice videos allow scrolling through slices of your data:

  1. Use the Windows leveling tab and the area box to properly contrast the region of interest
  2. Drag across the area of interest to set the contrast
  3. Adjust orientation (horizontal/vertical) as needed for better viewing
  4. To create a slice video:
  5. Go to the scenes view properties and turn off text annotations and legend if desired
  6. Navigate to the 2D animation tab
  7. Set the start and end slice numbers (e.g., from slice 200 to 300)
  8. Click play to preview the animation in the Dragonfly window
  9. To record the video:
  10. Select the recording option
  11. Choose a location to save the file
  12. Select the desired format (AVI or other)
  13. Adjust movie quality settings as needed

The resulting 2D slice video will reproduce everything visible in the viewer, including zoom level and contrast settings.

Preparing Data for 3D Videos [00:04:09]

For 3D videos, you can create more complex visualizations:

Setting Up Clipping

  1. Go to the shapes tool at the bottom of the main tab
  2. Create a box for clipping
  3. Adjust the box in 3D by clicking on the handles and moving with the left mouse button
  4. In the 2D windows, use the buttons to drag edges to the desired position
  5. To activate clipping:
  6. Click on the box
  7. Go to shape properties
  8. Select the image channel (chameleon head)
  9. Enable clipping
  10. Choose whether to clip inside or outside

Creating Segmentation for Bones

To visualize bones separately from the main structure:

  1. Create a manual segmentation using the manual thresholding method
  2. Name the segmentation (e.g., "bone")
  3. Enable 3D visualization
  4. For better bone rendering:
  5. Go to the segment tab
  6. Create a mesh for a nicer rendering
  7. Adjust smoothing if needed
  8. Customize the appearance:
  9. Make the bone more white
  10. Adjust contrast settings for better visualization

Setting the Background

Before starting the video, set an appropriate background:
- Choose a dark blue background with a gradient from top to bottom
- This helps provide 3D context to the visualization

Creating 3D Videos with Movie Maker [00:11:54]

The Movie Maker tool allows creation of complex 3D animations:

  1. Right-click on the 3D window and select "Show Movie Maker"

  2. Note: Any changes made in Movie Maker mode will be lost when exiting, so prepare everything beforehand

  3. Set up keyframes by:

  4. Positioning the model as desired for the first frame
  5. Adding a key by clicking the "Add Key" button
  6. Moving to a new position (rotating, zooming, etc.)
  7. Adding another key

  8. Each keyframe can be set to occur at specific times

  9. Example sequence for a chameleon visualization:

  10. First keyframe: Side view, zoomed in
  11. Second keyframe: Rotated upright
  12. Third keyframe: Complete rotation
  13. Fourth keyframe: Begin clipping to reveal internal structures
  14. Fifth keyframe: Change opacity settings to better view bone features

  15. Final keyframe: Change bone coloring for dramatic effect

  16. Adjust timing by dragging keyframes further apart for slower transitions

  17. Preview the animation by clicking the play button

  18. Export the animation:

  19. Click the "Export Animation" button
  20. Set video dimensions (1920×1080 for HD)
  21. Specify duration

  22. Save as video file with appropriate name

  23. To exit Movie Maker mode, right-click on the 3D window and select "Close Movie Maker"

Advanced Visualization Techniques [00:22:24]

For more complex visualizations:

  • Use multiple clipping boxes to reveal specific regions of interest
  • Apply different clipping settings to different structures (bones vs. soft tissue)
  • Visualize meshes alongside volume renderings
  • Change color lookup tables for dramatic effect
  • Adjust transparency settings to reveal internal structures

Note that rendering performance may be limited by your graphics hardware when visualizing many ROIs and images simultaneously, so try to limit what you have visualized at the same time.

Conclusion [00:24:06]

Creating videos in Dragonfly is a powerful way to showcase your data and findings. Whether making simple 2D slice videos or complex 3D animations with multiple structures and clipping planes, the software provides flexible tools to create professional visualizations. Remember to prepare your data and settings before entering Movie Maker mode, and take advantage of keyframes to create smooth transitions between different views and rendering styles.
